Templates Objects

The templates are using predefined objects stored in the Velocity Context by the generator.

Some objects are always available ($entity, $today, etc),
others are retrieved from existing objects ($attribute, $link).

All these objects are documented here :

  • $attribute
  • $const
  • $entity
  • $fn
  • $generator
  • $link
  • $loader
  • $project
  • $target
  • $today
